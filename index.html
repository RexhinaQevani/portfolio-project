Dictionary Project

I build this English dictionary project using react. It is a basic English dictionary where you can search for any English word and you will receive the phonetics of that word with an audio of its pronunciation, alongside all the possible meanings of the word with a detailed definition, an example and other synonyms. The dictionary is responsive so it can be used by any device and screen.

Weather Project

This is a basic weather website I built using react. It is responsive so you can check it out from any device.
